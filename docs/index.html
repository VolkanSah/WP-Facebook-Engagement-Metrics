Key Features

Automatic Updates

Metrics are refreshed every 3 hours using WordPress cron jobs, keeping your data current.

Dashboard Widget

Quickly view engagement trends right from your WordPress dashboard.

Performance Optimized

Access token caching reduces API requests and improves loading times.

Installation Guide

Step-by-Step Setup

1. Download the Plugin

Get the latest version from the GitHub repository.

2. Install and Activate

Upload the ZIP file via WordPress admin > Plugins > Add New.

3. Create Facebook App

Register a new app on the Facebook Developer Portal to get App ID and Secret.

4. Configure Settings

Enter your credentials in Settings > Facebook Engagement Metrics.

How It Works

Dashboard Widget

The plugin adds a widget to your WordPress dashboard showing:

  • Likes: Reaction counts on your Facebook links
  • Shares: Total number of times links have been shared
  • Comments: Comments associated with your links

You can also manually refresh metrics anytime.
